# 単語学習アプリ（GitHub Pages版）

端末内保存専用の英単語学習PWA。アカウント、クラウドDB、アクセス解析は使用しない。

## 公開手順

1. GitHubで公開リポジトリを1つ作成する。
2. このフォルダ内のファイルを、フォルダ構造を維持したままリポジトリ直下へアップロードする。
3. GitHubの `Settings` → `Pages` を開く。
4. `Deploy from a branch`、ブランチを `main`、フォルダを `/ (root)` に設定して保存する。
5. 発行されたURLをQRコード化して配布する。

## 重要な運用条件

- 配布後はリポジトリ名、GitHubユーザー名、独自ドメインなどを変更せず、URLを固定する。
- 学習データは「ブラウザ＋正確なサイトURL」ごとのIndexedDBに保存される。
- 同じ端末でもChromeとSafari、通常モードとプライベートモードではデータが別になる。
- ブラウザデータ削除、端末初期化・故障に備え、表示される通知に従ってJSONバックアップを保存する。
- PWAのインストールは任意。インストールしなくても利用できる。

## ファイル

- `index.html`: 画面
- `app.js`: 単語帳、問題、IndexedDB保存、バックアップ、OCR
- `service-worker.js`: アプリ本体のオフラインキャッシュ
- `manifest.webmanifest`: PWA設定
- `icons/`: アプリアイコン

## OCRについて

OCR画像自体はブラウザ内で処理され、オンラインDBへ保存されない。Tesseract.js本体と言語データを取得するため、OCRの初回実行時にはインターネット接続が必要になる場合がある。
